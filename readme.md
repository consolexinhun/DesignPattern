### 设计模式

#### 创建型模式：

1. 单例，一个类只能实例化一个对象
2. 原型，根据基类指针拷贝出派生类对象
3. 简单工厂，根据名称创建对应的对象，本质是基类指针指向派生类对象实现多态
4. 工厂方法，对象创建不再由工厂类进行，而是交由派生类执行
5. 抽象工厂，一个工厂不只生产一种产品，而是可以生产多种产品
